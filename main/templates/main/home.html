ML Based Road Extraction

Road View uses a machine learning model trained using U-Net architecture to make predictions. U-Net is a convolutional neural network and it works well even with fewer training samples and yields precise segmentation.
The output image is generated as a binary segmented image where the white pixels represent the road and the black pixels represent other information.
